# Materiais sociais TalkGlobal

Cada pasta corresponde a um artigo publicado.

Arquivos padrao:

- copy.txt: legenda/copy para Instagram.
- prompt-imagem.txt: prompt para criar a arte 1080x1350.
- info.txt: titulo, categoria, URL do artigo e status da imagem.

As imagens finais podem ser salvas na mesma pasta com nomes como imagem-instagram.png ou imagem-instagram.jpg.
