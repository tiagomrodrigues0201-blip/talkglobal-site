# Estratégia Amazon Afiliados - TalkGlobal

Objetivo: gerar cliques qualificados e buscar as primeiras 3 vendas Amazon em até 180 dias.

ID de associado: `talkglobal0a-20`

## Posicionamento

O TalkGlobal não deve parecer um catálogo de produtos. A monetização precisa nascer do contexto editorial:

- trabalho online;
- renda digital;
- home office;
- produtividade;
- ferramentas para vender e atender melhor;
- comunicação profissional.

A Amazon entra como recomendação útil, não como promessa de ganho rápido.

## Regras de links

Todos os links Amazon publicados devem usar a tag:

```txt
?tag=talkglobal0a-20
```

Formato final usado no site:

```txt
https://www.amazon.com.br/dp/ASIN?tag=talkglobal0a-20
```

Evitar publicar produtos sem ASIN real. Quando ainda não houver produto escolhido, manter o item fora do artigo em vez de deixar placeholder visível.

## Aviso obrigatório

Manter aviso claro nos artigos com links:

```txt
Este artigo pode conter links de afiliado. Podemos receber comissão sem custo adicional para você.
```

Também é recomendado manter no site uma declaração clara de associado Amazon, como:

```txt
Como associado da Amazon, o TalkGlobal pode receber comissão por compras qualificadas.
```

## Plano de 180 dias

### Dias 1-15: base

- Conferir se os links reais continuam ativos na Amazon.
- Priorizar produtos com alto apelo e preço acessível.
- Evitar produtos muito específicos ou caros no começo.
- Conferir se todos os links têm `tag=talkglobal0a-20`.
- Publicar os 3 artigos.
- Solicitar indexação no Search Console.

### Dias 16-45: tráfego inicial

- Criar 2 posts por semana no Instagram puxando para os artigos.
- Usar chamadas simples:
  - "Antes de comprar notebook para trabalhar online, veja isso."
  - "Seu home office pode estar te cansando sem você perceber."
  - "Ferramentas simples para começar online com mais organização."
- Colocar links dos artigos na bio ou em agregador de links.
- Reaproveitar trechos dos artigos como carrossel.

### Dias 46-90: otimização

- Ver quais artigos recebem mais cliques no Search Console.
- Trocar produtos com baixa atratividade.
- Adicionar novos artigos relacionados:
  - "Melhor microfone para gravar vídeos simples"
  - "Itens baratos para melhorar seu home office"
  - "Livros para quem quer começar a vender online"
- Criar links internos entre artigos.

### Dias 91-180: escala

- Criar uma página central: `/amazon.html` ou `/ferramentas.html`.
- Agrupar recomendações por perfil:
  - para freelancer;
  - para vendedor online;
  - para home office;
  - para criador de conteúdo.
- Publicar comparativos editoriais sem prometer "melhor absoluto".
- Medir cliques e substituir recomendações fracas.

## Métrica principal

O primeiro objetivo não é comissão alta. É gerar clique qualificado.

Métricas para acompanhar:

- impressões no Google;
- cliques nos artigos;
- CTR dos artigos;
- cliques nos botões Amazon;
- vendas atribuídas no painel de Associados;
- produtos que geram mais saída.

## Produtos com maior chance de clique

Priorizar categorias amplas e úteis:

- acessórios de notebook;
- mouse sem fio;
- suporte para notebook;
- ring light/luminária;
- microfone simples;
- headset;
- planner;
- livros;
- mochila;
- cadeira ergonômica.

- itens sem relação com o artigo;
- links soltos sem explicação editorial.

## Tom editorial

O texto deve parecer recomendação útil de portal, não anúncio.

Usar:

- "vale observar";
- "pode ajudar";
- "faz sentido para quem";
- "antes de comprar";
- "procure modelos com";
- "comece pelo que mais atrapalha sua rotina".

Evitar:

- "compre agora";
- "imperdível";
- "garantido";
- "o melhor do mercado";
- "você vai ganhar dinheiro com isso".
